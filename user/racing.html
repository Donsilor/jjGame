【多彩种测试房（可修改名字）定位竞猜规则】

【游戏介绍】

 ：是经国家财政部批准，由北京福利彩票发行承销的中国福利彩其中一种；每期开奖车号共十个,每个车号除了整合玩法，第1~10名、冠亚和值、冠亚组合分别为一个竞猜组，大小/单双/龙虎、车道数字、车道非数字、冠亚大小、冠亚特码、特码区段。

【相关资料】
【开奖官网】www.bwlc.net
【开奖时间】09:07~23:57 每5分钟一期，一天共179期。

玩法1 【1~10名猜大小单双】

第一名～第十名车号：开出之号码大于或等于6为大，小于或等于5为小。开出的号码偶数为双，号码奇数为单。
■竞猜奖励：含本1.94倍
■下注限额：10-1000
■单期上限：3000
■格式：名次/大小单双/金额
例：12345/双/100 = 1~5车道买双各100 = 总500

玩法2 【1~10名猜车号】

每一个车号为一竞猜组合，开奖结果「竞猜车号」对应所猜「车道」视为中奖，其余情形视为不中奖。
■竞猜奖励：含本9.7倍
■下注限额：10-300
■单期上限：3000
■格式：名次/号码/金额
例：12345/89/20 = 1~5车道买8号、9号各买20 = 总200
例：1357/890/20 = 1.3.5.7车道买8号、9号、10号各买20 = 总240

玩法3 【1~10名猜组合】

竞猜内容为「大单」「小双」「小单」「大双」，共4种。
■竞猜奖励：
「大单」、「小双」含本0倍，
「小单」、「大双」含本0倍。
■下注限额：10-0
■单期上限：0
■格式：名次/组合/金额
例：890/大单/50 = 8.9.10车道大单各买50 = 总150

玩法4 【1~5名猜龙虎】

(1)第一名vs第十名，(2)第二名vs第九名，(3)第三名vs第八名，(4)第四名vs第七名，(5)第五名vs第六名，前比后大为龙，反之为虎。
■竞猜奖励：含本1.94倍
■下注限额：10-1000
■单期上限：3000
■格式：名次/号码/金额
例：123/龙/100 = 1~3车道买龙各100=总300

玩法5 【猜冠亚号码】

猜冠军及亚军车号，每次竞猜2个号码，顺序不限。
■竞猜奖励：含本0倍
■下注限额：10-100
■单期上限：1000
■格式：组/号码/金额
例：组/56/50 = 5号.6号车在冠亚军(顺序不限) = 总50
例：组/19.10/100 = 1.9号车或1.10号车在冠亚军(顺序不限) = 总200

玩法6 【冠亚和值(特码)猜大小单双】

冠军车号+亚军车号 = 冠亚和值 = 特码 = 数字3~19
冠亚和值大于或等于12为大，小于或等于11为小。开出的号码偶数为双，号码奇数为单。
■竞猜奖励：
「大」、「双」含本2.1倍。
「小」、「单」含本1.7倍。
■下注限额：10-1000
■单期上限：200000
■格式：和(特)/大小单双/金额
例：和双100 = 「冠亚和」的双100
例：和大100 = 「冠亚和」的大100

玩法7 【冠亚和值(特码)猜数字】

「冠亚和值」为「特码」可能出现的结果为3~19，竞猜中对应「冠亚和值」数字的视为中奖，其余视为不中奖。
■竞猜奖励：
    3.4.18.19    含本40倍
    5.6.16.17    含本20倍
    7.8.14.15    含本12倍
    9.10.12.13    含本10倍
    11             含本8倍
■下注限额：10-500
■单期上限：2000
■格式：和(特)/数字/金额
例：和5.6.7/100 = 竞猜「冠亚和」的值为5或6或7各100 = 总300
例：和56710/100 = 竞猜「冠亚和」的值为5或6或7或10各100 = 总400
